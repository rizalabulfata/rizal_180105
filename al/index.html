    Numerical Solution of System Linier Algebraic Solution

    Sistem Persamaan Aljabar Linear¶

    dari contoh persamaa diatas dimana: a1, a2,…, an, dan b adalah konstanta dan x1, x2,…, xn adalah variabel yang tidak diketahui.

    Pengertian¶

    System persamaan linear adalah himpunan beberapa persamaan linear dalam variabel x1, x2,…, xn.

    Barisan bilangan s1, s2,… sn adalah solusi system persamaan linear apabila x1= s1 , x2= s2 … xn =sn memenuhi setiap persamaan linear dalam system.

    Ada tiga kemungkin dalam penyelesaian SPL yaitu mempunyai hanya 1 penyelesaian, mempunyai banyak penyelesaian, atau bahkan tidak mempunyai penyelesaian:

    1. hanya mempunyai 1 penyelesaian seperti : x1=2, and x2=1

    2. mempunyai banyak penyelesaian seperti : = x1=1, x2=2, and x3=-1

    3. tidak mempunyai penyelesaian

    untuk penyelesaian tersebut dapat menggunakan System persamaan Linear dalam Augmented Matriks, yang nantinya akan banyak metode dalam menggunakannya.

    contoh: Augmented matriksnya

    Metode untuk menyelesaikan SPL adalah dengan mengubah system yang ada menjadi system yang lebih sederhana. Secara umum langkah langkahnya diberikan di bawah ini.

    1. Mengalikan satu atau lebih dari persamaan yang ada dengan suatu pengali
    2. Mempertukarkan letak persamaan
    3. Menambah suatu persamaan dengan persamaan yang lain

    Mencari penyelesaian sistem persamaan linier¶

    untuk mencari penyelesain Sistem Persamaan Linier metodenya terdapat 2 macam, yaitu :

    • Metode Eliminasi
    • gauss
    • gauss jordan
    • Metode Iterasi
    • seidel
    • jacobi

    Metode Elminiasi Gauss¶

    Konsepnya didasarkan pada gagasan mereduksi matriks yang diperbanyak menjadi bentuk yang cukup sederhana sehingga sistem persamaan tersebut bisa diselesaikan dalam bentuk substitusi.

    contoh :

    1. Menuliskan bentuk augmented Matriks

    2. Hilangkan kolom pertama di bawah diagonal. Gantikan baris 2 dengan baris 2-2xbaris 1

    3. ganti baris 3 dengan baris 3-baris 1

    4. Hilangkan kolom kedua dbawah diagonal. Gantikan baris 3 dengan baris 3-3xbaris2

    5. gunakan substitusi untuk mendapatkan penyelesaian

    Berikut perhitungan menggunakan python dengan tambahan modul numpy :

    import numpy as np

    a = [[2,1,-1, 8], [-3,-1,2,-11], [-2,1,2,-3]]
n = len(a[0])-1
x = np.zeros(len(a[0]))
for i in range(n):
    if a[i][i] == 0.0:
        sys.exit('tidak boleh dibagi 0')

    for j in range(i+1, n):
        ratio = a[j][i]/a[i][i]

        for k in range(n+1):
            a[j][k] = a[j][k] - ratio * a[i][k]

x[n-1] = a[n-1][n]/a[n-1][n-1]

for i in range(n-2,-1,-1):
    x[i] = a[i][n]

    for j in range(i+1,n):
        x[i] = x[i] - a[i][j]*x[j]

    x[i] = x[i]/a[i][i]

print("hasilnya : ")
print(f"x = {x[0]}\ny = {x[1]}\nz = {x[2]}")

    hasilnya : 
x = 2.0
y = 3.0
z = -1.0

    Metode Eliminasi Gauss Jordan¶

    Eliminasi Gauss-Jordan adalah pengembangan dari eliminasi Gauss yang hasilnya lebih sederhana lagi. Caranya adalah dengan meneruskan operasi baris dari eliminasi Gauss sehingga menghasilkan matriks yang Eselon-baris. Ini juga dapat digunakan sebagai salah satu metode penyelesaian persamaan linear dengan menggunakan matriks. Metode eliminasi Gaus/Gaus-Jordan dilakukan dengan menggunakan langkah-langkah pada Operasi Baris Elementer (OBE). Sebelum melakukan OBE, bentuk umum dari sistem persamaan linear dirubah menjadi matriks yang diperbesar.

    contoh : jawab :

    1. Menuliskan bentuk augmented Matriks

    2. tukarkan baris ke 1 dengan baris ke 3

    3. kalikan (-3) pada baris 1 kemudian jumlahkan ke baris 2

    4. kalikan (-2) pada baris 1 kemudian jumlahkan ke baris 3

    5. kalikan 1/3 pada baris 2

    6. kalikan (-2) pada baris 2 kemudian jumlahkan ke baris 3

    7. kalikan 3 pada baris 3

    8. kalikan 11/3 pada baris 3 kemudian jumlahkan baris 3

    9. kalikan (-2) pada baris 3 kemudian jumlahkan ke baris 1

    10. kalikan (-1) pada baris 2 kemudian jumlahkan ke baris 1

    11. Hasil dari operasi baris tersebut kemudian dirubah kembali menjadi bentuk umum sistem persamaan linear.

    12. Sistem persamaan linear yang diperoleh dapat ditulis sebagai berikut. Jadi penyelesaian dari sistem persamaan linear tersebut adalah

    Metode Iterasi Seidel¶

    Metode interasi Gauss-Seidel adalah metode yang menggunakan proses iterasi hingga diperoleh nilai-nilai yang berubah-ubah. Metode iterasi Gauss-Seidel dikembangkan dari gagasan metode iterasi pada solusi persamaan tak linier. rumus dan metode gauss seidel: contoh : jawab :

    #fungsi
f1 = lambda x,y,z: (7+1-1)/4
f2 = lambda x,y,z: (21+4*x+z)/8
f3 = lambda x,y,z: (14-2*x-y)/5

x0 = 0
y0 = 0
z0 = 0
count = 1
e = float(input('toleransi error: '))
condition = True

while condition:
    x1 = f1(x0,y0,z0)
    y1 = f2(x1,y0,z0)
    z1 = f3(x1,y1,z0)
    print(f"iterasi-{count} : x = {x1} | y = {y1} | z = {z1} ")
    e1 = abs(x0-x1);
    e2 = abs(y0-y1);
    e3 = abs(z0-z1);

    count += 1
    x0 = x1
    y0 = y1
    z0 = z1

    if e1 > e and e2 > e and e3 > e:
        condition = False

print("\nhasilnya : ")
print(f"x = {x1}\ny = {y1}\nz = {z1}")

    toleransi error:  0.001
iterasi-1 : x = 1.75 | y = 3.5 | z = 1.4 

hasilnya : 
x = 1.75
y = 3.5
z = 1.4

    Metode Iterasi jacobi¶

    Metode Jacobi merupakan salah satu metode penyelesaian sistem persamaan linear (baca: matriks) berdimensi banyak (baca: lebih dari 2). Untuk matriks dengan dimensi kecil (kurang atau sama dengan dua), lebih efektif diselesaikan dengan aturan eliminasi atau metode Cramer.
